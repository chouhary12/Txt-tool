# Telegram Tools Bot 🤖

## Features
- TXT → HTML
- HTML → TXT
- Extract Links
- Compare Links
- Domain Changer
- Merge TXT
- Split Links
- Text → TXT file

## Deploy (Render)

1. Upload to GitHub
2. Go to Render
3. New Web Service → Connect repo
4. Add ENV:
   BOT_TOKEN = your_token
   WEBHOOK_URL = https://your-app.onrender.com
5. Deploy 🚀
